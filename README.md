# Spam Email Classifier

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Make sure the dataset file is in the same folder:
   Spam_Email_Dataset.xlsx

3. Run the model:
   python model.py

## Output
- Prints accuracy, precision, recall, F1-score
- Saves confusion_matrix.png
- Shows predictions for 3 sample emails
